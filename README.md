# Utvecklarportfölj – Frontend 1

Detta projekt är en enkel utvecklarportfölj skapad som inlämningsuppgift i kursen Frontend 1.  
Portföljen är byggd med HTML, CSS och JavaScript.

## Vad sidan innehåller

- Hero-sektion med namn, titel och navigation
- About Me med porträttbild och presentation
- Skills-sektion med teknologier
- Portfolio med exempelprojekt (placeholder)
- Testimonials med referenser
- Kontaktformulär och knapp för att ladda ner CV
- Footer med länkar till GitHub och LinkedIn

## Tekniker som används

- **HTML** för struktur och innehåll
- **CSS** med Flexbox och Grid för layout samt enklare animationer
- **JavaScript** för interaktivitet vid scroll, visning av navigation och animationer

## Metoder och principer

- HTML, CSS och JavaScript är uppdelade i separata filer
- Samma CSS-klasser återanvänds för animationer (t.ex. `reveal` och `is-visible`)
- JavaScript används för att lägga till och ta bort CSS-klasser istället för att styra design direkt

## Hur projektet körs

Projektet är statiskt och kan köras lokalt genom att öppna `index.html` i en webbläsare.
